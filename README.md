Comando para criar um projeto no React:

npx create-next-app@latest

----------------------------------------

1) Digitar "y" para prosseguir
2) Digite um nome para o projeto (enter para nome padrão "my-app")
3) Selecionar configurações customizadas (customize settings)
4) Selecionar a opção "No" para não usar o TypeScript
5) Selecionar a opção "None" para não usar nenhum Linter
6) Selecionar a opção "Yes" para ativar o React Compiler
7) Selecionar a opção "No" para não usar o Tailwid CSS
8) Selecionar a opção "Yes" para usar o código dentro do source directory (`src/ directory`)
9) Selecionar a opção "Yes" para ativar o App Router
10) Selecionar a opção "No" para não customizar o import de abreviações (import alias)
